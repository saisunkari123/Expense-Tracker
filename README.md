
# Smart Expense Tracker Dashboard

A responsive financial management web application designed to help users track their income and expenses with real-time insights.

## 🚀 Key Features
* **Real-time Calculations:** Automatically calculates total balance, income, and expenses as data is entered.
* **Category-based Tracking:** Organize spending into specific categories for better financial clarity.
* **Data Visualization:** Integrated **Chart.js** to provide interactive spending pattern charts.
* **Persistent Storage:** Utilizes **Browser Local Storage** to ensure data remains available after page refreshes without a backend dependency.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Libraries:** Chart.js
* **Storage:** Web Storage API (LocalStorage)

## 📖 How to Run
1. Clone this repository.
2. Open `index.html` in any modern web browser.
